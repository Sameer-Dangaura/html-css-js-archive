> CSS Pseudo-classes
---------------------

* pseudo => means not genuine;


> What are Pseudo-classes?
:
# A pseudo-class is used to define a special state of an element.

For example, it can be used to:

- Style an element when a user moves the mouse over it
- Style visited and unvisited links differently
- Style an element when it gets focus
- Style valid/invalid/required/optional form elements


Syntax:
-------
The syntax of pseudo-classes:
____________________________
selector:pseudo-class {
  property: value;
}
____________________________



> The :hover pseudo-class   (some are done in html part also)
--------------------------
:
# The :hover pseudo-class is used to style effects when a user moves the mouse over it, style visited and unvisited links differently.


e.g-1:
___________________
a:link {
  color: #FF0000;
}
a:visited {
  color: #00FF00;
}
___________________


e.g-2: 
____________________
a:hover {
  color: #FF00FF;
}
____________________


e.g-3:
____________________
div:hover {
  background-color: blue;
}
________________________


e.g-4: Simple Tooltip Hover (do visit it's named folder)
:
Hover over a <div> element to show a <p> element (like a tooltip).
