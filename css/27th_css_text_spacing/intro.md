> Text Indentation
-------------------
:
The text-indent property is used to specify the indentation of the first line of a text:

# indent meaning => start (a line of text) or position (a block of text) further from the margin than the main part of the text.

> White Space
--------------
:
The white-space property specifies how white-space inside an element is handled.

This example demonstrates how to disable text wrapping inside an element:
                          ______________________________________________
Example:

p {
  white-space: nowrap;  
}

# This nowrap(means: no wrap) is also used in "47th_css_float_example/equal_height_boxes_by_flexbox"   to disable text wrapping inside container   to make text flexible or, you can say    to prevent the items inside the flexbox container from wrapping onto the next line.
