Without clear: below elements also floats

Div 1
Div 2 -> Notice that div2 is after div1 in the HTML code. However, since div1 floats to the left, the text in div2 flows around div1.

With clear: cleared element will appear below it on the web page.

Div 3
Div 4 -> Here, clear: left; moves div4 down below the floating div3. The value "left" clears elements floated to the left. You can also clear "right" and "both".
