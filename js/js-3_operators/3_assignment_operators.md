> Types of JavaScript Operators:

2) JavaScript Assignment Operators
: 
 Assignment operators assign values to JavaScript variables. 


Operator	Example	    Same As

=	        x = y	    x = y
+=	        x += y	    x = x + y
-=	        x -= y	    x = x - y
*=	        x *= y	    x = x * y
/=	        x /= y	    x = x / y
%=	        x %= y	    x = x % y
**=	        x **= y	    x = x ** y  (Exponent)


# Note: I think all operators above mentioned are same as other programming languages like C, C++. Excluding exponent.
