> Type Conversion in JavaScript
:
- >  Type conversion in JavaScript is an algorithm that allows us to manipulate the function's data types from one to another. 
            Also, sometimes, the    JavaScript compiler 
 automatically does these conversions and ensures that the function gets the valid inputs according to the operator used.


# Types of type conversion in JavaScript
:
1) Implicit conversion
2) Explicit conversion



1) Implicit conversion (Automatic conversion)
:
- Implict conversion in JavaScript refers as when
converting the values of one data type 
to another    automatically .


2) Explicit conversion (Type casting)
:
- The conversion of one data type to another according to our need and using built-in methods is called as explicit type casting in JavaScript.
